Есть
1) калман, который сильно фильтрует даже идеальные значения
2) графики ускорений 
3) Примитивная стабилизация, которая жутко неустойчивая. Демпфирующих звеньев нет.

Надо
1) подписи к графикам, единицы измерения, автомасштабирование графиков
2) оценка удаления от точки старта по радиус вектору. 
3) Вывод на экран позиции вертолета в ГСК. 
4) Импорт внешних данных из таблиц с замерами. Модель должна повторить то что мы делали с датчиками
5) следование камеры за вертолетом по принципу "прицепа с люфтом в плоскости ХУ", чтобы вертолет 
не находился жестко закрепленным в центре кадра. Камера должна смотреть паралельно оси Z
